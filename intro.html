Introduction

Radiens supports Python 3.7 and newer.

Installation

Install the most recent Jinja version using pip:

$ pip install radiens

Basic Usage

Radiens currently provides two clients AllegoClient and VidereClient that can be used to interact with their respective servers Allego and Videre. They can be imported and used via the following:

from radiens import AllegoClient, VidereClient

allego = AllegoClient()
videre = VidereClient()
